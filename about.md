This page explains how the demo site is organized.

## What This Template Includes

- a home page with intro content
- topic hubs
- articles with cross-links
- a search page
## How to Use

1. Replace demo text with your own
2. Create new hubs and articles
3. Connect pages via `hub`, `related`, and wiki links
4. Publish through the workflow.

This is a neutral starter set with no personal or brand-specific references.
